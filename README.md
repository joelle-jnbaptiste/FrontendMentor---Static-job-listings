## ğŸ”® About The Project

This project is a solution to the **Job Listings with Filtering** challenge from **Frontend Mentor**.

The goal was to build a **job board interface** where users can:
- View job listings with a responsive layout
- See hover states for interactive elements
- Filter job offers dynamically based on selected categories

This project emphasizes:
- Angular component architecture
- State-driven filtering logic
- Separation of concerns using services
- Responsive and scalable UI styling

## ğŸª„ Getting Started

This project was built using **Angular CLI** and deployed on **GitHub Pages**.

### ğŸ“˜ Prerequisites

- Node.js
- npm
- Angular CLI

### ğŸ§™ Installation

1. Clone the repository:

       git clone https://github.com/joelle-jnbaptiste/FrontendMentor---Static-job-listings.git

2. Install dependencies:

       npm install

3. Run the development server:

       ng serve

4. Open your browser at `http://localhost:4200`
