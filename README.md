## ğŸ¯ About The Project

This project is a solution to the **Static Job Listings** challenge from **Frontend Mentor**.

The goal was to build a **job board interface** that allows users to:

- Browse job listings in a responsive layout
- Filter job offers dynamically using selected tags
- See hover states for interactive elements
- Clear filters to return to the full list

This project emphasizes:

- Angular component architecture
- State-driven filtering logic
- Separation of concerns using services
- Responsive and scalable UI styling

It demonstrates the ability to implement **dynamic UI behavior** on top of a static dataset.

## âš”ï¸ Getting Started

This project was built using **Angular CLI** and can be run locally.

### Prerequisites

- Node.js
- npm
- Angular CLI

### Installation

1. Clone the repository:

       git clone https://github.com/joelle-jnbaptiste/FrontendMentor---Static-job-listings.git

2. Install dependencies:

       npm install

3. Run the development server:

       ng serve

4. Open your browser at:

       http://localhost:4200
